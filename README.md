# BGN Messaging 

Documentation: apidoc/index.html

No UI is provided to test this system.  

### Install
```
git clone https://github.com/Sam-Ogden/BGN-Messaging.git
cd BGN-Messaging/BGNMessage
npm install
```

### Run app localhost:3000
```
npm start
```

### Test app (mocha+chai)
```
 npm test
```
